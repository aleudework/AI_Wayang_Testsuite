USER_REQUEST: Make a Wayang Plan that based on tenancy types with housing_tenancy = 1 finds the total number of move outs for 2024 for each tenancy type. Beside the total number for each tenancy, also the tenancy type description should be a part of the result. You don't need any sortering or ordering. Output in a textfile.
SQL_QUERY:
SELECT 
    t.tenancy_types_id,
    t.tenancy_description,
    SUM(f.sum_of_move_outs) AS total_moveouts_2024
FROM factmoveouts f
JOIN tenancy_type t
    ON f.tenancy_types_id = t.tenancy_types_id
WHERE f.move_out_year = 2024
  AND t.housing_tenancy = 1
GROUP BY 
    t.tenancy_types_id,
    t.tenancy_description
ORDER BY 
    total_moveouts_2024 DESC;
